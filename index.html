Character Co-occurrence in Shakespearean Drama

Play: Order:

This application visualizes the degree to which characters in Shakespeare's plays appear together.

Each colored cell represents two characters that appeared in the same scene, and darker cells indicate characters that co-occurred more frequently.

Use the drop-down menus to select a different play, reorder the matrix, and explore the data.

Built with data from ProQuest's Chadwyck Healey Literature Online Collections.
